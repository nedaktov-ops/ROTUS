# ROTUS Obsidian Vault - The Rotten Brain of POTUS

This vault contains verified quotes from Donald Trump with full source attribution.

## Structure

- `Quotes/` - Individual quote notes with metadata
- `Sources/` - Source documentation (FactCheck, PolitiFact, Snopes, etc.)
- `People/` - Related people and entities
- `Topics/` - Thematic organization
- `Dashboards/` - Overview notes and MOC (Map of Content)

## Quote Note Template

Each quote note includes:
- Full quote text
- Date, time, location
- Platform/source
- Context and backdrop
- Verification status
- Multiple source links
- Confidence score

## Usage

This vault serves as:
1. Source of truth for the ROTUS app
2. Research workspace for quote verification
3. Knowledge graph for entity relationships
4. Export source for web/mobile apps

## Graphify Integration

Run `/graphify .` to generate knowledge graph from this vault.
